Centos 更换yum源

介绍

由于CentOS7默认的yum源是国外的，
导致我们使用yum下载软件的下载速度不是很理想，这时候我们就需要将yum源更换成国内的源。

可联网

1.首先我们先对系统本身的yum源进行备份

2.下载国内yum源配置文件到/etc/yum.repos.d/CentOS-Base.repo

3.清理yum缓存，并生成新的缓存

4. 更新yum源检查是否生效

内网环境

1. 下载ISO，上传到服务器上

2. 备份本地源

3. 挂载镜像

4. 开机挂载（非必要）

5. 写入源

6. 测试安装软件
